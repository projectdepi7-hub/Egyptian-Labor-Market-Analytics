## ğŸ“Œ **Project Overview**

This project analyzes **real Egyptian labor market data** to understand how educational backgrounds, vocational training, and non-formal programs relate to employment outcomes. Using the latest available dataset, the analysis examines the alignment between labor market needs and the skills supplied by different educational and training tracks.
The project provides insights into demographics, household characteristics, employment conditions, income patterns, and skill gaps across Egypt. The goal is to support evidence-based planning that benefits policymakers, employers, educators, training institutions, households, and youth.
